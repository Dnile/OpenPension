# Open Pension

Just an example of reading xlsx files using an included parser.

## Requirments

npm modules:
*   node-zip
*   xml2js
    
## Usage

```shell
node test.js <file> <sheet> <cell>
```

Where `<file>` is xlsx file, `<sheet>` is one-based sheet number and `<cell>` is the cell in standart Excel format(e.g. E1).
